A fundamental concept of modern network communication is the "Internet Protocol" address. They provide a system to uniquely identify endpoints on a network. Often it is helpful to know the IP address of your device, so this site provides that information if avaliable.

Multiple Internet websites exist to provide IP address information already, but this site is setup differently. It runs locally on campus and provides additional benefits.

  • Devices on campus without Internet connectivity can access this site.
  • VPN users will see the address used on campus as part of the default split-tunnel VPN configuration.
  • Additional information about campus addresses is provided if available.

Further information is available from the help site in the Determine IP address help article.
