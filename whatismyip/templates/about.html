About This Tool

This site is hosted on the campus network and provided by Information Technology Services as a network diagnostic tool. The IP addresses reported are those visible to campus applications such as addresses provided by the campus VPN service. Refer to the Determine Client IP Address help article for site questions or comments.

A fundamental concept of modern network communication is the "Internet Protocol" address. It provides a system to uniquely identify endpoints on a network. Often it is helpful to know the IP address of your device, so this site provides that information if available.

Multiple Internet websites exist to provide IP address information already, but this site is set up differently. It runs locally on campus and provides additional benefits.

  • Devices on campus without Internet connectivity can access this site.
  • VPN users will see the address used on campus as part of the default split-tunnel VPN configuration.
  • Additional information about campus addresses is provided if available.

Further detailed network information is available through the Network Information Tool (NIT). Access to NIT is restricted to authorized users.

Open Source

The source code for this tool is available on GitHub. It is written in Python using the Flask framework and is released under the MIT License.

The tool was built for the UNC Chapel Hill campus environment but is designed to be adapted by other higher education institutions. Campus network ranges, DNS security filtering settings, and institution branding are all configurable without changing code. Integrations with network management systems such as Infoblox IPAM and Extreme Networks NAC are optional — the core IP address and geolocation features work for any deployment.

If your institution runs a similar tool or adapts this one, we'd love to hear about it.
